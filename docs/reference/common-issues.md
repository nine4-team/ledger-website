# Common Issues

## I Cannot See A Project

Check whether you are viewing Active or Archived projects.

If you still cannot see it, ask an admin to confirm your account access.

## I Cannot See A Fee, Invoice, Or Financial Total

Your financial access may be limited. Ask an admin to check your role and financial access settings.

## A Transaction Appears In Review

Open the transaction and check:

- Project or inventory assignment.
- Budget category.
- Vendor/source.
- Amount.
- Date.
- Linked items.

## An Item Is Missing From A Report

Check whether the item has the needed:

- Project assignment.
- Space assignment.
- Price.
- Budget category.
- Invoice membership, if the report is invoice-related.

## A Budget Total Looks Wrong

Check:

- Whether the transaction category is correct.
- Whether a return reduced the spend.
- Whether a transaction is canceled.
- Whether the category is excluded from overall budget totals.
- Whether your financial access hides fee data.

## An Item Was Marked Returned But Still Looks Active

Changing an item status does not always complete the return. Use the return workflow to create or link the return transaction.

## Search Cannot Find Something

Try:

- Alternate vendor/source names.
- SKU instead of item name.
- Project name instead of client name.
- Clearing filters.

The record may also be hidden by permissions.
