Item Statuses

Item status describes where an item is in its lifecycle.

To Purchase

The item has been identified but not purchased yet.

Purchased

The item has been purchased and is active.

To Return

The item is marked for return but the return may not be fully processed yet.

Returned

The item has been returned.

Sold

The item has been sold or moved under a sale flow.

Notes

Statuses can be corrected. If an item was marked incorrectly, update the status or process the correct movement.

Changing status alone may not complete a financial return. Use the return workflow when the item is actually returned.
