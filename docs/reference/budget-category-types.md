# Budget Category Types

Budget category type controls how a category behaves in forms, budgets, and reporting.

## Itemized

Use for physical products that should be tracked as individual items.

Examples:

- Furnishings.
- Accessories.
- Rugs.

Itemized categories usually support item entry and inventory movement workflows.

## Expense

Use for non-itemized costs that do not need individual item records.

Examples:

- Delivery.
- Install.
- Fuel.
- Storage.

## Fee

Use for company revenue or service fees.

Examples:

- Design Fee.
- Retainer.
- Project Management Fee.

Fee categories may be protected by financial access settings.

## General

General is a broad or legacy type. It may behave like an expense category in some workflows.
