Admin

Archive Or Delete Projects

Use this when a project should no longer appear in the active project list.

Prefer Archive

Archive is the normal choice for completed or inactive projects. It hides the project from active work while preserving:

  • Items.
  • Transactions.
  • Spaces.
  • Notes.
  • Budgets.
  • Invoices.
  • Reports.

Delete Is Destructive

Deleting a project removes the project record. It may leave related records without a project home.

Use delete only when the project was created by mistake and your team is sure the history is not needed.

Steps To Archive

  1. Open the project.
  2. Open the project options menu.
  3. Choose Archive Project.
  4. Confirm.

Steps To Unarchive

  1. Go to the Archived project list.
  2. Open the project.
  3. Open the project options menu.
  4. Choose Unarchive Project.
  5. Confirm.
