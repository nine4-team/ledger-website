# Manage Space Templates

Use this when you need reusable room or location templates with checklists.

## Before You Start

You need admin access.

Know:

- Template name.
- Default notes, if any.
- Checklist names and checklist items.

## Steps

1. Go to **Settings**.
2. Open **Presets**.
3. Choose **Templates**.
4. Add or edit a space template.
5. Add checklists and checklist items.
6. Save the template.

## How Templates Are Used

When someone creates a space from a template, Ledger copies the template notes and checklists into the new space. Checklist items start unchecked.

## Common Issues

- Editing a template does not necessarily update spaces that were already created from it.
- Archive old templates instead of deleting them when you want to preserve history.
