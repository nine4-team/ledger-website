# Manage Vendors

Use this when you need to maintain vendor/source presets for transactions and items.

## Before You Start

You need admin access.

## Steps

1. Go to **Settings**.
2. Open **Presets**.
3. Choose **Vendors**.
4. Add, edit, or remove vendor names.
5. Save changes.

## Good Vendor Naming

Use names that employees will search for consistently.

Examples:

- Amazon.
- Wayfair.
- Local Receiving Warehouse.
- Business Inventory.

Avoid duplicate spellings for the same vendor if possible.

## Common Issues

- Search and filters work better when vendor names are consistent.
- If transactions are split across multiple spellings, consider standardizing the vendor name.
