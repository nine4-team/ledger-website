# Review Queue

The Review area is a work queue for transactions that need attention.

## What Appears In Review

Review can surface transactions that are:

- Unassigned.
- Associated with inventory.
- Associated with a project.
- Missing details.
- Pending a decision.

The list is organized by bucket so you can focus on one area at a time.

## How To Use Review

Use Review to:

- Find transactions that need cleanup.
- Open a transaction and confirm its project or inventory assignment.
- Search pending transactions.
- Sort review work by date.
- Resolve unassigned or incomplete records.

## Empty State

If Review says everything is caught up, there are no pending transactions in the review queue.
