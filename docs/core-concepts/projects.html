Core Concepts

Projects

Projects are the main workspace for client work in Ledger. A project contains the items, transactions, spaces, notes, budgets, invoices, and reports for one client engagement.

What A Project Contains

Inside a project, you can work with:

  • Items: physical products assigned to the project.
  • Transactions: records of purchases, expenses, fees, sales, and returns.
  • Spaces: rooms or locations where items belong.
  • Notes: project notes.
  • Finances: budget tracking, billing, invoices, and reports.

Active And Archived Projects

Projects can be active or archived.

  • Active projects appear in the main project list.
  • Archived projects are hidden from normal active work but keep their data.

Archive projects when the work is complete but the history should remain available. Deleting a project is destructive and should be rare.

Budgets In Projects

Each project can have its own enabled budget categories and budget amounts. Project cards show budget previews so you can quickly see which categories are active or near their limits.
