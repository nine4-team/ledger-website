# Budgets

Budgets help the team track project spending by category.

## Budget Categories

Budget categories are account-level presets, such as:

- Furnishings.
- Install.
- Design Fee.
- Storage and Receiving.

Each project chooses which categories are enabled and how much budget is allocated to each category.

## Category Types

Categories can behave differently depending on type:

- **Itemized**: physical items are tracked individually.
- **Expense**: non-itemized costs.
- **Fee**: revenue or money owed to the business.
- **General**: legacy or broad category behavior.

## How Spend Is Calculated

Project budget progress is based on transactions.

In general:

- Purchases and expenses increase spend.
- Returns reduce spend.
- Canceled transactions are excluded.
- Fee categories show money received or expected differently from spending categories.

## Pinned Budget Categories

Project cards and project detail pages can highlight important budget categories. If categories are pinned, those categories are shown first.
