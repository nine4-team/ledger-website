# Search

Search helps you find records across Ledger when you do not know exactly where they live.

## What Search Finds

Search can find:

- Items.
- Transactions.
- Spaces.

Results are separated into tabs by result type.

## When To Use Search

Use Search when:

- You know an item name, SKU, vendor, or source.
- You need to find a transaction by vendor or notes.
- You need to find a space by name.
- You are not sure which project contains the record.

## Bulk Actions From Search

Some search result types allow selection and bulk actions. For example, you may be able to select multiple items and update their status, space, transaction, or project assignment.

Be careful with bulk actions. Confirm you are selecting the right records before applying changes.
