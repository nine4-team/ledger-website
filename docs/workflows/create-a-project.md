# Create A Project

Use this when you need to add a new client project to Ledger.

## Before You Start

You may need admin or project creation access.

Have these details ready:

- Project name.
- Client name.
- Optional description.
- Optional hero image.
- Budget categories for the project.
- Budget amounts, if known.

## Steps

1. Go to **Projects**.
2. Tap the create button.
3. Choose **New Project**.
4. Enter the project name, client name, description, and optional hero image.
5. Select the budget categories this project should use.
6. Add a new category if the needed category does not exist.
7. Enter budget amounts for the selected categories.
8. Save the project.

## What Happens Next

Ledger creates the project and enables the selected budget categories. If you added a hero image, the image may finish uploading in the background.

The project appears in the active project list.

## Common Issues

- If the project does not appear immediately, check whether you are viewing Active or Archived projects.
- If a category is missing, ask an admin to confirm whether it is archived or needs to be created.
- If the image does not appear right away, it may still be uploading.
