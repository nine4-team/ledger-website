# Export Transactions

Use this when you need a CSV export of project transactions.

## Before You Start

Open the project whose transactions you need to export.

If you only want certain transactions, apply search or filters in the Transactions area first.

## Steps

1. Open the project.
2. Go to **Transactions** if you want to export the filtered transaction list.
3. Open the project options menu.
4. Choose **Export Transactions**.
5. Select the fields to include.
6. Export the file.
7. Share or save the generated CSV using the system share sheet.

## What Happens Next

Ledger creates a CSV export from the selected transaction set.

## Common Issues

- If exported data looks incomplete, check active filters.
- If fee or invoice-related data is missing, your financial access may be limited.
- If you need all project transactions, export from project options without narrowing the transaction list.
