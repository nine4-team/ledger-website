# Capture Item Quick Drafts

Use this when you need to quickly capture item photos before you know all the item details.

## Before You Start

Open the project, inventory area, or transaction context where the draft belongs.

Use quick drafts when:

- You are taking field photos.
- Item details are incomplete.
- A reviewer will clean up the item later.
- You need to group photos before creating full item records.

## Steps

1. Open the relevant project or item context.
2. Go to **Items**.
3. Tap the add button.
4. Choose **Item Quick Draft**.
5. Capture or select photos.
6. Add any known context, such as name, source, or notes.
7. Save the draft.

## What Happens Next

Ledger creates an Item Quick Draft. It appears in the Item Quick Drafts section for that context.

The draft does not count as a full item until someone converts or merges it.

## Common Issues

- If a draft belongs to inventory, mark it as inventory-related before conversion.
- If duplicate photos were captured, merge the draft into an existing item instead of creating a duplicate item.
- If a draft was captured by mistake, delete the draft.
