# Review Pending Transactions

Use this when the Review area shows transactions that need attention.

## Before You Start

Know which bucket you want to review:

- Unassigned.
- Inventory.
- A specific project.

## Steps

1. Go to **Review**.
2. Choose the bucket you want to work through.
3. Use search or sort if the list is long.
4. Open a transaction.
5. Check project assignment, vendor/source, amount, category, notes, and linked items.
6. Make the needed correction.
7. Return to Review and continue until the bucket is clear.

## What Happens Next

Resolved transactions should disappear from the pending review list or move to the correct bucket.

## Common Issues

- If a transaction is unassigned, confirm whether it belongs to inventory or a project.
- If the category is wrong, update the transaction before relying on budget totals.
- If a transaction should not count, cancel or correct it according to your team's process.
