# Add An Item To A Project

Use this when you are ready to create a full item record in a project.

If you only have photos or partial information, use [Capture Item Quick Drafts](capture-item-quick-drafts.md) first.

## Before You Start

Open the project where the item belongs.

Have any available item details ready:

- Photos.
- Name.
- SKU.
- Vendor/source.
- Notes.
- Purchase price.
- Project price.
- Market value.
- Quantity.
- Space.
- Transaction.

## Steps

1. Open the project.
2. Go to **Items**.
3. Tap the add button in the Items area.
4. Choose **Item**.
5. Add photos and basic details.
6. Add pricing, quantity, status, transaction, and space details if known.
7. Save the item.

## What Happens Next

Ledger creates the item in the project. Photos may upload in the background.

If you linked a transaction or space, the item appears in those related areas too.

## Common Issues

- An item needs either a name or at least one image.
- If the right transaction is missing, create or correct the transaction first.
- If the right space is missing, create the space first or leave the item unassigned.
- If the item is actually business inventory, add it from inventory or use the inventory workflow instead.
