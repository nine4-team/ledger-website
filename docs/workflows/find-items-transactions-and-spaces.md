# Find Items, Transactions, And Spaces

Use this when you need to find something but do not know where it lives.

## Steps

1. Go to **Search**.
2. Type a name, SKU, vendor, source, note, or space name.
3. Wait for results to appear.
4. Choose the result type:
   - Items.
   - Transactions.
   - Spaces.
5. Open the record you need.

## Tips

- Search broader terms first, then narrow down.
- Try vendor/source names for transactions.
- Try SKU or item name for items.
- Try room names for spaces.

## Bulk Actions

Some search results can be selected for bulk actions. Use this carefully and confirm the selected records before applying changes.

## Common Issues

- If you cannot find a record, it may be archived, hidden by permissions, or stored under a different vendor/source name.
- If results update slowly, clear the search and try a more specific term.
