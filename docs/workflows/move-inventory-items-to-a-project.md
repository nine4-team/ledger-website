# Move Inventory Items To A Project

Use this when business inventory items should become part of a client project.

## Before You Start

Know:

- Which inventory items are moving.
- Destination project.
- Budget category for the batch.

All selected items in the batch use the same destination category. If items need different categories, move them in separate batches.

## Steps

1. Find the inventory items.
2. Select one or more items.
3. Open the item action menu or bulk actions menu.
4. Choose **Sell to Project**.
5. Select the destination project.
6. Select the budget category for the batch.
7. Confirm the move.

## What Happens Next

Ledger moves the items into the destination project and creates a purchase-from-inventory transaction.

The destination project's budget spend increases for the selected category.

## Common Issues

- Inventory items do not keep project budget categories while in inventory.
- If the destination category is wrong, correct it before confirming.
- If multiple categories are needed, split the items into separate moves.
- If you only need to fix a transaction link within the same project, use a correction/move action instead of selling to a project.
