# Generate Client Reports

Use this when you need a client-facing report from project data.

## Report Types

Ledger supports reports such as:

- Invoice report.
- Client summary report.
- Property management report.

## Before You Start

Open the project and confirm the data is accurate:

- Items.
- Spaces.
- Prices.
- Transactions.
- Invoices.
- Budget categories.

## Steps

1. Open the project.
2. Go to **Finances**.
3. Open **Reports**.
4. Choose the report type.
5. Review the report on screen.
6. Print or share the report if needed.

## Common Issues

- If items are missing from a property management report, confirm they are assigned to spaces.
- If a client summary looks incomplete, check item prices and categories.
- If an invoice report looks wrong, open the invoice and review its lines.
